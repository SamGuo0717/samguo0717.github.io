Vim 常用命令参考

模式切换

命令 功能
i 进入插入模式（当前位置）
I 进入插入模式（行首）
a 进入插入模式（当前位置后）
A 进入插入模式（行尾）
o 新建下一行并进入插入模式
O 新建上一行并进入插入模式
Esc 退出插入模式到普通模式
: 进入命令模式
v 进入可视模式
V 进入行可视模式

光标移动

命令 功能
h 左移
j 下移
k 上移
l 右移
w 下一个单词开头
b 上一个单词开头
e 单词结尾
0 行首
$ 行尾
gg 文件开头
G 文件结尾
Ctrl+f 向下翻页
Ctrl+b 向上翻页

编辑操作

命令 功能
x 删除当前字符
dd 删除当前行
yy 复制当前行
p 粘贴到下方
P 粘贴到上方
u 撤销
Ctrl+r 重做
dw 删除单词
cw 修改单词
cc 修改整行

搜索替换

命令 功能
/text 向下搜索text
?text 向上搜索text
n 下一个匹配
N 上一个匹配
:s/old/new 替换当前行第一个old为new
:s/old/new/g 替换当前行所有old为new
:%s/old/new/g 替换全文所有old为new

文件操作

命令 功能
:w 保存
:q 退出
:wq 保存并退出
:q! 强制退出不保存
:w filename 另存为
:e filename 打开文件

行号与跳转

命令 功能
:set nu 显示行号
:set nonu 隐藏行号
:n 跳转到第n行
nG 跳转到第n行

窗口操作

命令 功能
:sp 水平分割窗口
:vsp 垂直分割窗口
Ctrl+w w 切换窗口
Ctrl+w q 关闭当前窗口
